Ultimate Tab for WooCommerce is a plugins for unlimited custom tab for WooCommerce. You can add unlimited custom tab content and tab title.

== Description ==
Ultimate Tab for WooCommerce is a WooCommerce plugin. Using this plugin, you can create an unlimited tab in WooCommerce product single page. Sometimes fixed and few tab is not enough to display product data correctly. You can add an unlimited tab in WooCommerce product single page with your custom title.

* Add Ultimate Tab in WooCommerce Single Page.
* Support Special Character in Description.
* Support Direct HTML in Description
* Translate Ready

== Installation ==

1. Go to Dashboard > Plugins > Add New, then search “ultimate-tab-for-woocommerce” and click Install Now
2. Upload 'ultimate-tab-for-woocommerce’ to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. This is a WooCommerce based plugins so you have to install WooCommerce.
